发表于 2024-12-11 更新于 2024-12-22 分类于 GitHub

问题描述

在使用 Git 推送代码到 GitHub 时，推送经常失败，但带宽测试显示网络正常，上传和下载速度都没问题。经过排查发现，Git 默认不会通过本地代理，即使代理设置为全局模式也无效。这导致 GitHub 的访问不稳定。
